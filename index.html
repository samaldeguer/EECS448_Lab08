Lab 8

  • Exercise 1: Password Validator
    • This page allows the user to enter a password two times in order to validate it. If the passwords do not match or the password is less than 8 characters long, an error message will pop up.

  • Exercise 2: Slide show!
    • This page displays 5 images in a slide show manner, with previous and next buttons. When needed, the images will wrap from the last image to the first and from the first to the last. All images stay the same size as well.

  • Exercise 3: Personal Profile
    • This page displays a personal profile of the creator, Sam Aldeguer. Included are a profile picture, a short blurb, my favorite sites hyperlinked, and an embedded video of one of my vlogs. Site is stylized using CSS.

  • Exercise 4: CSS Manipulation
    • This page displays a paragraph of dummy text, and fields that allow you to change the colors of the background and border of the text, as well as the width of the border. All inputs must be entered in RGB format.
